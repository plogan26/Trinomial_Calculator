Fairly straight forward, insert the value of the coefficient of x^2 into the first box and the value of the constant into the second box. After you've entered two valid integers, click "Factor." The factors for the entered values appear below the input boxes and all of the potential equations will display in the right-most display box. These results are all of the possible formatulations based on the factors of the input values. Scroll through the list to find the middle-constant value you're looking for.

Two notes:

1) Only enter positive integers into the input boxes. There will be a handful of variations of the value you're looking for in the results, find the one with the correct signs. If a value in the trinomial equation is represented as 'x^2' or just 'x,' input a 1 for that value.

2) For lack of coding, you'll notice that all of the calculated formulations have positive operators. Negative values are expressed as (a + -b).
